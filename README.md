# 📸 Photo Clone API

A RESTful Spring Boot application for uploading, storing, retrieving, downloading, and deleting photos using **Spring Data JDBC** and an **H2 In-Memory Database**.

This project demonstrates file handling, binary data storage, REST API development, and database integration using modern Spring technologies.

---

## 🚀 Features

✅ Upload images using multipart file requests

✅ Store image metadata and binary content in a database

✅ Download stored images by ID

✅ Retrieve single or multiple photo records

✅ Delete photos from the database

✅ Interactive API testing with Swagger UI

✅ Lightweight H2 in-memory database for development

## 🗄️ Database Schema

The application uses Spring Data JDBC and initializes the schema at startup.

```sql
CREATE TABLE IF NOT EXISTS photos (
    id INT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    folder_address VARCHAR(255) NOT NULL,
    data BLOB,
    content_type VARCHAR(50)
);
```

# 🔌 API Endpoints

## 📤 Upload Photo

Upload an image file to the database.

**Endpoint**

```http
POST /add
```

**Content-Type**

```text
multipart/form-data
```

### Request Parameters

| Parameter | Type          | Required |
| --------- | ------------- | -------- |
| data      | MultipartFile | Yes      |

### Response

```json
{
  "id": 1,
  "folderAddress": "sample-image.png",
  "contentType": "image/png"
}
```

---

## 📋 Get All Photos

Retrieve metadata for all stored photos.

**Endpoint**

```http
GET /photos
```

### Response

```json
[
  {
    "id": 1,
    "folderAddress": "sample-image.png",
    "contentType": "image/png"
  }
]
```

---

## 🔍 Get Photo Metadata By ID

Retrieve photo information using its unique identifier.

**Endpoint**

```http
GET /photos/{id}
```

### Example

```http
GET /photos/1
```

### Response

```json
{
  "id": 1,
  "folderAddress": "sample-image.png",
  "contentType": "image/png"
}
```

### Error

```http
404 NOT FOUND
```

---

## 📥 Download Photo

Download the original image file.

**Endpoint**

```http
GET /download/{id}
```

### Example

```http
GET /download/1
```

### Response

Returns the raw image bytes with the original content type.

```http
Content-Type: image/png
```

---

## ❌ Delete Photo

Delete a photo from the database.

**Endpoint**

```http
DELETE /photos/{id}
```

### Example

```http
DELETE /photos/1
```

### Response

```http
200 OK
```

# ▶️ Running the Application

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/photo-clone-api.git
cd photo-clone-api
```

---

## 2. Build the Project

```bash
mvn clean install
```

---

## 3. Run the Application

```bash
mvn spring-boot:run
```

The application will start on:

```text
http://localhost:8080
```
